### Student Tasks
- [ ] In `Program.cs` add and use a named policy:
```csharp
const string DevCors = "DevCors";
builder.Services.addCors(o => o.AddPolicy(DevCors, p => p
  .WithOrigins("http://localhost:5173")
  .AllowAnyHeader()
  .AllowAnyMethod()));
app.UseCors(DevCors);
```
- [ ] Run API; later verify by calling from the React app.
